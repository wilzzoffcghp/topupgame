API Documentation

NexaTop API memungkinkan kamu mengintegrasikan layanan top up game ke platform kamu sendiri. Semua harga sudah termasuk markup Rp1.000 per transaksi.

ℹ️ API ini hanya tersedia untuk Reseller terdaftar. Daftar sekarang dengan biaya Rp15.000 permanen.
Base URL
https://yourdomain.vercel.app/api/v1
Autentikasi

Semua request harus menyertakan API Key di header X-Api-Key. API Key didapatkan setelah mendaftar sebagai reseller.

⚠️ Jangan bagikan API Key kamu kepada siapapun. Simpan dengan aman.
Header
Salin
X-Api-Key: rsl_your_api_key_here
Content-Type: application/json
🎮 Game
GET /game/list
List Game

Mendapatkan semua game yang tersedia beserta icon dan kategorinya.

JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/game/list', {
  headers: { 'X-Api-Key': 'rsl_your_api_key' }
});
const data = await response.json();
Contoh Response
JSON
Salin
{
  "success": true,
  "data": [
    {
      "game_name": "Mobile Legends",
      "game_slug": "mobile-legends",
      "category": "game",
      "icon_url": "https://cdn.example.com/mobile-legends"
    }
  ]
}
GET /game/:slug/detail
Detail Game & Produk

Mendapatkan daftar produk/nominal yang tersedia untuk game tertentu. Harga sudah termasuk markup Rp1.000.

Path Parameters
ParameterTipeKeterangan
slug*stringSlug game dari endpoint list (contoh: mobile-legends)
JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/game/mobile-legends/detail', {
  headers: { 'X-Api-Key': 'rsl_your_api_key' }
});
const data = await response.json();
Contoh Response
JSON
Salin
{
  "success": true,
  "data": {
    "game_name": "Mobile Legends",
    "game_slug": "mobile-legends",
    "products": [
      {
        "product_code": "ML1048",
        "product_name": "Mobile Legends 1048 Diamonds",
        "price": 196000,
        "description": "Top Up Diamond Mobile Legends Instan"
      }
    ]
  }
}
GET /game/:slug/product/:code
Cek Produk

Mengecek ketersediaan dan harga produk tertentu sebelum melakukan order.

JavaScript
Salin
const response = await fetch(
  'https://yourdomain.vercel.app/api/v1/game/mobile-legends/product/ML1048',
  { headers: { 'X-Api-Key': 'rsl_your_api_key' } }
);
📦 Order
POST /game/order
Buat Order

Melakukan top up untuk game tertentu. Sertakan target_zone hanya jika game memerlukan Zone ID (contoh: Mobile Legends).

Request Body
FieldTipeKeterangan
product_code*stringKode produk dari endpoint detail game
target*stringUser ID / akun tujuan top up
target_zonestringZone ID (opsional, hanya untuk game tertentu)
JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/game/order', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Api-Key': 'rsl_your_api_key'
  },
  body: JSON.stringify({
    product_code: "ML1048",
    target: "136216325",
    target_zone: "2685" // hapus jika tidak perlu zone
  })
});
Contoh Response
JSON
Salin
{
  "success": true,
  "message": "Transaksi berhasil diterima dan diproses.",
  "data": {
    "order_id": "ZXT-1779607267-101",
    "product_name": "Mobile Legends - 1048 Diamonds",
    "target": "136216325",
    "price": 196000,
    "status": "processing"
  }
}
POST /game/status
Cek Status Order

Mengecek status transaksi berdasarkan Order ID. Status bisa: success, processing, atau failed.

JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/game/status', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Api-Key': 'rsl_your_api_key'
  },
  body: JSON.stringify({ trx_ref: "ZXT-1779607267-101" })
});
Contoh Response
JSON
Salin
{
  "success": true,
  "data": {
    "trx_ref": "ZXT-1779607267-101",
    "product_name": "Mobile Legends - 1048 Diamonds",
    "price": 196000,
    "status": "success",
    "sn": "9812492194821",
    "completed_at": "2024-01-15 11:30:15"
  }
}
💳 Deposit
POST /deposit/create
Buat Deposit QRIS

Membuat permintaan deposit saldo via QRIS. Minimal Rp10.000.

Request Body
FieldTipeKeterangan
amount*integerJumlah nominal deposit dalam Rupiah (min: 10000)
JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/deposit/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Api-Key': 'rsl_your_api_key'
  },
  body: JSON.stringify({ amount: 50000 })
});
Contoh Response
JSON
Salin
{
  "success": true,
  "deposit": {
    "id": "DP-1002",
    "total_payment": 50000,
    "qr_image": "https://link-to-qris.png",
    "status": "pending"
  }
}
POST /deposit/status
Cek Status Deposit

Mengecek apakah deposit sudah dibayar. Lakukan polling setiap 5–10 detik hingga status success.

JavaScript
Salin
const response = await fetch('https://yourdomain.vercel.app/api/v1/deposit/status', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Api-Key': 'rsl_your_api_key'
  },
  body: JSON.stringify({ deposit_id: "DP-1002" })
});
Contoh Response
JSON
Salin
{
  "success": true,
  "status": "success",
  "message": "Pembayaran diverifikasi"
}
